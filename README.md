# Week 3 – TestNG Automation Assignment

## 📌 Project Overview
This project demonstrates the usage of **TestNG** with Java and Maven for automation testing.  
It covers:
- TestNG annotations
- XML configuration (`testng.xml`)
- Page Object Model (POM) framework design
- Reporting with Extent Reports
- Data-driven testing (Excel + JSON)
- Selenium Grid setup for remote execution
